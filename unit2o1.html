1-7 Describe which are the appropriate file types for websites

The appropriate file types to use for websites would be the following whether it is uploading images, videos or music. I used images so the file-type I used for my About me page and the background of my website was PNG and JPG.

JPEG: One of the most common one to use, because the format is user friendly for all types of photographs.
PNG: If you take a lot of screenshots, this format is perfect to use and for other images that dont't contain a lot of color.
GIF: This format is for those who have a lot of animated graphics, because it gives people a chance to show their artistic side on their site.
JPG: This is a digital image format, thats contains compressed image data. This format is mainly used for sharing photos and other images on the internet.

Every file-type is different, depending what you are uploading, such as for uplaoding videos the file types would be Webm, flv, mov, qt and much more and for audio you have wav, mp3, aac and so on. For more information. themeisle.com/blog/best-image-format/!
